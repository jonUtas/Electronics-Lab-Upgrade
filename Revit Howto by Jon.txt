Howto Add change project revisions:
select floor plan
view (tab) -> Sheet Composition -> revisions
 

Howto Update revisions from project revisions:
	Go to sheet. 
	in browers pane select families filter
	search for Title_Blocks_A1_Metric
	select Title_Blocks_A1_Metric
	look at properties
	should be able to select "revisions on sheet"
	tick revisions you want displayed.
